Daily Workflow

Edit files → Upload via cPanel → SSH in → git add . && git commit
Save your work (do this after every change)
git add . git commit -m "what you changed"

See What's Happening

What files have changed?
git status
What exactly changed in those files?
git diff
See your commit history
git log --oneline
See what changed in the last commit
git diff HEAD~1

Undo Mistakes

Discard changes to ONE file (restore to last commit)
git checkout -- filename.php

Example: git checkout -- index.php

Discard ALL changes (restore everything to last commit)
git checkout -- .
⚠️ Warning: checkout throws away uncommitted changes. Make sure you really want to lose them.
