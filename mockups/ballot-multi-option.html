Ranked choice ballot — drag to reorder or use arrows. Instant runoff eliminates the lowest option and redistributes votes until a winner meets the threshold. The grip handle (☰) and arrows both work for reordering. Mobile users use arrows exclusively.

Component Legend

New components shown in this mockup:

• Vote type badges — Yes/No (green), Multi-Choice (blue), Ranked (purple)
• Round indicator — dots showing current position in multi-round voting
• Threshold progress bar — gold marker at threshold, fill color changes when met
• Quorum badge — shows participation vs requirement
• Option cards — checkbox-select with source post reference and vote bars
• Rank cards — draggable with grip handle + arrow fallback
• Facilitator panel — gold-themed controls (only shown to facilitator role)
• Round results — between-round view with merge suggestions
• Declaration banner — ratified group mandate at top of group page
