DISCUSS & DRAFT -- Video Narration Script
=========================================
Video: discuss-and-draft-walkthrough.webm
Duration: ~67 seconds

Timestamps are approximate -- sync to on-screen captions.


REQUIRED TOOLS
--------------
| Tool                   | Purpose                          | Install                          |
|------------------------|----------------------------------|----------------------------------|
| Node.js                | Runs Playwright                  | nodejs.org                       |
| Playwright             | Records headless browser session | npm install (devDependency)      |
| PowerShell Speech API  | Generates TTS narration audio    | Built into Windows               |
| ffmpeg                 | Merges video + audio             | winget install ffmpeg            |
| XAMPP                  | Local web server for recording   | Already installed                |


HOW TO CREATE A NARRATED VIDEO WALKTHROUGH
-------------------------------------------

Step 1: Write the walkthrough script
   - File: scripts/generate-video-walkthrough.js
   - Define each step: page navigation, mouse movement, clicks, typing, captions
   - Playwright controls a headless Chromium browser

   MOUSE & VISUAL OVERLAYS (important!):
   - injectCursor(page) — injects a white SVG arrow cursor that follows mouse moves
     * MUST be called after every page.goto() — navigation destroys the DOM
     * Without it, the video shows no mouse movement (headless has no real cursor)
   - slowMove(page, x, y, steps) — smoothly moves cursor to coordinates
   - moveToElement(page, selector) — moves cursor to center of element
   - clickFlash(page, x, y) — shows a gold ring animation at click point
   - clickElement(page, selector) — moves to element, flashes, clicks
   - hoverElement(page, selector, duration) — moves to element, shows tooltip
   - selectDropdown(page, selector, value) — animated fake dropdown overlay
   - caption(page, text, duration) — centered caption bar at bottom of viewport
   - All timing is logged to a JSON file for audio sync

Step 2: Record and narrate (automated — one command does both)
   - Start XAMPP (Apache must be running on localhost)
   - Run:
     SITE_URL=http://localhost/tpb2 AUTH_USER_ID=1 node scripts/generate-video-walkthrough.js <name>
   - Available walkthroughs: discuss-and-draft, profile, getting-started, all
   - The script automatically:
     1. Records silent video with cursor overlays + captions (.webm)
     2. Saves timing log (.json) mapping each caption to its timestamp
     3. Generates TTS audio clips via edge-tts (en-US-JennyNeural voice)
     4. Inserts silence gaps between clips to sync with video timing
     5. Merges video + audio into final .webm with ffmpeg

   Voice: en-US-JennyNeural (female, natural) via edge-tts Python package
   - pip install edge-tts (if not installed)
   - Rate: -10% (slightly slower for clarity)
   - All TTS generation is automatic — no manual PowerShell needed

Step 3: Verify and link
   - Play the final .webm to check cursor, captions, and narration sync
   - Add a "link" field to the guide JSON (step 1 of the guide):
     "link": {
       "url": "/help/videos/<name>-walkthrough.webm",
       "label": "Watch the video walkthrough",
       "target": "_blank"
     }

   ffmpeg merge details (handled automatically by the script):
   - -c:v copy = keep video as-is (fast, no re-encode)
   - -c:a libopus = encode audio as Opus (small, web-optimized)
   - No -shortest flag — audio plays to completion even if video is shorter

Step 6: Link from the guide page
   - Add a "link" field to the guide JSON (help/data/talk.json):
     "link": {
       "url": "/help/videos/discuss-and-draft-walkthrough.webm",
       "label": "Watch the video walkthrough",
       "target": "_blank"
     }
   - Guide renderer (help/guide.php) renders it as a gold link
   - Opens in a new browser tab, WebM plays natively full-window

Step 7: Push to staging
   - git add + commit + push
   - git pull on staging server


TO CREATE ANOTHER VIDEO
-----------------------
1. Add a new async function walkthroughXxx(context) in generate-video-walkthrough.js
   - Use injectCursor(page) after every page.goto()
   - Use slowMove/moveToElement for visible mouse movement
   - Use clickElement (not page.click) for gold click flash
   - Use caption() for on-screen text (auto-logged for TTS timing)
   - Use hoverElement for tooltip reveals
   - Use selectDropdown for animated dropdown selection
2. Register it in the walkthroughs object at the bottom of the file
3. Run: SITE_URL=http://localhost/tpb2 AUTH_USER_ID=1 node scripts/generate-video-walkthrough.js <name>
   - Records video + generates JennyNeural narration + merges — all in one step
4. Add link to step 1 of the relevant guide JSON in help/data/
5. Push to staging
